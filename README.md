- **Set-up**
```Bash

pkg install git -y

```

```Bash
cd $HOME
rm -rf AutoM-FB
git clone --depth=1 https://github.com/Mr-SxR/AutoM-FB.git
cd AutoM-FB
git pull
python AutoM-FB.py
```
